## Project Configuration

- **Language**: TypeScript
- **Package Manager**: npm
- **Add-ons**: prettier, eslint, tailwindcss

---

This is a website for the Curacao Rugby Federation.
A visitor should be able to get all the information they need about
their story, training times, signing up, a feed for facebook/instagram, and coming events.

It should be built in the latest stable sveltekit.
It should allow an admin to change its content.
It should use Supabase as it database and authentication.
